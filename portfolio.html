Portfolio

Here you can see some of the projects I've done (the wider boxes). These 3 projects below were all for educational purposes.

Recreate an Old Retro Game

Together with 5 other students we've recreated an old retro game. This was created with C# and MonoGame in Visual Studio. Features: online/offline multiplayer, highscores, play against the computer. This is created in 10 weeks.

Artificial Intelligence

Classification, Reinforcement Learning, Path Planning algorithms, Bayesian Networks and much more.

Multithreaded Programming

Such as programming on the GPU with OpenCL.

Information Retrieval and Data Mining

This included building a search engine for cars with techniques such as term frequency, inverse document frequency and Jaccard index.

Webshop

My first client-server application: a webshop where you can browse, filter and buy products (not for real, but the idea remains the same). You could also create an account and login with that account. Created with Jade, CSS3, JavaScript, jQuery, Node.js, Express.js and AJAX. Created in approximately 3 weeks in groups of two.

Datastructures

Learned all kinds of search algorithms such as quicksort, counting sort, selection-sort and bitwise operations and how to program efficiently.

Small game in Haskell

Created in 4/5 weeks in groups of two. Haskell is not the most ideal for game development, but this was a mandatory requirement.

Software Design
